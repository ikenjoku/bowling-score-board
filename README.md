### Installation

Clone repo, install the dependencies and start the app.

```sh
$ git clone https://github.com/ikenjoku/bowling-score-board.git
$ cd bowling-score-board
$ yarn install
$ yarn start
```
### Tests

Clone repo, install the dependencies.

```sh
$ yarn test
```
